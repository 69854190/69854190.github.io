Stream流

java stream流
java核心
发布日期:   2023-04-11
文章字数:   2.3k
一、概述

1、Stream流的思想
　　Java8新增的Stream是为了解放程序员操作集合(Collection)时的生产力,结合了Lambda表达式，简化集合、数组的操作。Stream流就好像一个高级的迭代器，但只能遍历一次，也可以理解为流水线，在流的过程中对流中的元素进行操作，比如过滤长度为５的字符串，获取中以张开头的字符串等。
2、使用步骤
　　先获取到一条Stream流（流水线），并把数据放上去，然后利用Stream流中的API进行各种操作，这些操作包含过滤、转换等中间方法，也就是方法调用完成后还可以调用其他方法，可以有多个每次返回一个新的流，可进行链式操作，还包含统计、打印等终结方法，只能放在最后且只有一个。

二、创建流

　　如果是单列集合直接使用Collection中的默认方法stream()创建流，如果是数组则使用Arrays中的静态方法stream()创建流，也可以使用Stream中的of()方法（底层调用的还是Arrays.stream()），双列集合无法直接使用stream流，需要通过keySet()或者entrySet转成单列集合才能进行流的操作。对于非集合非数组的一堆同类型的数据也可以创建流，调用Stream的of()方法。 

//1、单列集合创建流
      List<String> arrayList=new ArrayList<>();
      arrayList.add("张三");
      arrayList.add("李四");
      arrayList.add("王麻子");
      Stream<String> stream1 = arrayList.stream();
      //2、数组创建流
      String[] arr=new String[]{
          "周扒皮","杜子腾","梅驰趵"};
      Stream<String> stream2 = Arrays.stream(arr);
//使用of()方法数组必须是引用类型的，如果传递基本数据类型会把整个数组当成一个元素。
      Stream<String> stream3 = Stream.of(arr);
      //3、双列集合创建流
      HashMap<String, Integer> map = new HashMap<>();
      map.put("英语",78);
      map.put("数学",90);
      map.put("语文",88);
      //创建键的Stream流
      Stream<String> stream4 = map.keySet().stream();
      //创建键值对的Stream流
      Stream<Map.Entry<String, Integer>> stream5 = map.entrySet().stream();
      //4、创建一对零散数据的流
      Stream<Integer> stream6 = Stream.of(1, 2, 3, 4, 5, 6, 7, 8, 9);
      //使用终结方法遍历打印数据
      stream1.forEach(list-> System.out.println(list));
三、Stream流的中间方法

1、过滤
通过 filter() 方法可以从流中筛选出我们想要的元素。
filter() 方法接收的是一个 Predicate（Java 8 新增的一个函数式接口，接受一个输入参数返回一个布尔值结果）类型的参数，因此，我们可以直接将一个 Lambda 表达式传递给该方法，比如说 element -> element.contains(“王”) 就是筛选出带有“王”的字符串。每用中间方法一次就会会返回新的Stream流，原来的Stream流只能使用一次，修改Stream流中的数据，不会影响原来集合或者数组中的数据。

List<String> arrayList=new ArrayList<>();
       arrayList.add("张三");
       arrayList.add("李四");
       arrayList.add("王五");
       arrayList.add("赵六");
       arrayList.add("张得美");
       //过滤出以张开头的字符串，并且遍历打印
       //forEach() 方法接收的是一个 Consumer（Java 8 新增的一个函数式接口，接受一个输入参数并且无返回的操作）类型的参数，
       // 类名 :: 方法名是 Java 8 引入的新语法，System.out 返回 PrintStream 类,也就是相当于打印操作   arrayList.stream().filter(list>list.startsWith("张")).forEach(System.out::println);

2、获取元素、跳过元素、去重
limit(long maxSize)：获取前几个元素。
skip(long n): 跳过前几个元素。
distinct():元素去重，依赖hashCode()和equals方法，所有操作自定义JavaBean时记得重写这两个方法。
concat()：合并ab两个流为一个流，尽量保持ab数据类型的一致，否则合并的新流的数据类型就是ab数据类型的共同父类。

List<String> arrayList=new ArrayList<>();
     arrayList.add("张三");
     arrayList.add("还有");
     arrayList.add("李四");
     arrayList.add("励志");
     arrayList.add("成为");
     arrayList.add("java");
     arrayList.add("高手");
     //获取“李四” "励志" "成为"
     //思路：获取前5个元素，再跳过前两个元素
     arrayList.stream().limit(5).skip(2).forEach(System.out::println);
     //对arr数组中的元素去重
     String[] arr=new String[]{"张三","李四","张三","王五","李四"};
     Arrays.stream(arr).distinct().forEach(System.out::println);
     //合并上面用到的两个流
     System.out.println("----------------------");    Stream.concat(arrayList.stream(),Arrays.stream(arr)).forEach(System.out::println);

3、映射
如果想通过某种操作把一个流中的元素转化成新的流中的元素，比如转换流中的数据类型，可以使用 map() 方法。 

List<String> arrayList=new ArrayList<>();
       arrayList.add("张三");
       arrayList.add("李四");
       arrayList.add("王五");
       arrayList.add("赵六六");
       //获取字符串长度将其通过map转换成Integer类型的流
       Stream<Integer> stream = arrayList.stream().map(list -> list.length());
       stream.forEach(System.out::println);
       //以上代码还可以写成
       arrayList.stream().map(String::length).forEach(System.out::println);

map() 方法接收的是一个 Function（Java 8 新增的一个函数式接口，接受一个输入参数 T，返回一个结果 R）类型的参数，此时参数 为 String 类的 length 方法，也就是把 Stream 的流转成一个 Stream 的流。

四、Stream流的终结方法

1、匹配
Stream 类提供了三个方法可供进行元素匹配，它们分别是：
anyMatch()，只要有一个元素匹配传入的条件，就返回 true。
allMatch()，只有有一个元素不匹配传入的条件，就返回 false；如果全部匹配，则返回 true。
noneMatch()，只要有一个元素匹配传入的条件，就返回 false；如果全部匹配，则返回 true 

List<String> arrayList=new ArrayList<>();
     arrayList.add("张三");
     arrayList.add("李四");
     arrayList.add("王五六");
     arrayList.add("赵六六");
     //任意一个元素包含六就为true
     boolean anyMatch = arrayList.stream().anyMatch(list -> list.contains("六"));
     System.out.println(anyMatch);//true
     //必须全部以张开头才为true
     boolean allMatch = arrayList.stream().allMatch(list -> list.startsWith("张"));
     System.out.println(allMatch);//false
     //只要有一个满足条件，也就是只要有一个元素包含六就为false
     boolean noneMatch = arrayList.stream().noneMatch(list -> list.contains("六"));
     System.out.println(noneMatch);//false

2、组合
reduce() 方法的主要作用是把 Stream 中的元素组合起来，它有两种用法：
Optional reduce(BinaryOperator accumulator)
没有起始值，只有一个参数，就是运算规则，此时返回 Optional。
T reduce(T identity, BinaryOperator accumulator)
有起始值，有运算规则，两个参数，此时返回的类型和起始值类型一致。 

Integer[] ints = {0, 1, 2, 3};
    List<Integer> list = Arrays.asList(ints);
    //没有初始值
    Optional<Integer> optional = list.stream().reduce((a, b) -> a + b);
    //运算规则可以是 Lambda 表达式（比如 (a, b) -> a + b），也可以是类名::方法名（比如 Integer::sum)
    Optional<Integer> optional1 = list.stream().reduce(Integer::sum);
    System.out.println(optional.orElse(0));
    System.out.println(optional1.orElse(0));
   //带初始值
    int reduce = list.stream().reduce(6, (a, b) -> a + b);
    System.out.println(reduce);
    int reduce1 = list.stream().reduce(6, Integer::sum);
    System.out.println(reduce1);

3、常用终结方法
forEach(Consumer action)：遍历，在前面的例子中都有使用，这里就不再举例子了。
count(): 统计。
toArray():收集流中的数据，放到数组当中。
collect(Collector collector):收集流中的数据，放到集合中。collect()会在后面的转换流中详细介绍。 

List<String> arrayList=new ArrayList<>();
     arrayList.add("张三");
     arrayList.add("李四");
     arrayList.add("王五");
     arrayList.add("赵六");
     arrayList.add("赵七");
     arrayList.add("赵八");
     //统计流中元素个数
     long count = arrayList.stream().count();
     System.out.println(count);//6
     //收集流中的数据转换为数组
     //将以赵开头的元素收集为一个新的数组
     Object[] array = arrayList.stream().filter(list -> list.startsWith("赵")).toArray();
     System.out.println(Arrays.toString(array));
五、转换流

既然可以把集合或者数组转成流，那么也应该有对应的方法，将流转换回去——collect() 方法就满足了这种需求。
Collectors 是一个收集器的工具类，内置了一系列收集器实现，比如说 toList() 方法将元素收集到一个新的 java.util.List 中；比如说 toCollection() 方法将元素收集到一个新的 java.util.ArrayList 中；比如说 joining() 方法将元素收集到一个可以用分隔符指定的字符串中。 

List<String> list = new ArrayList<>();
     Collections.addAll(list, "张三-男-12", "李四-男-17", "王五-女-18", "赵六-男-22");
     //将集合中所有性别为男收集到一个新的Lis集合
     List<String> stringList = list.stream().filter(element -> "男".equals(element.split("-")[1]))
             .collect(Collectors.toList());
     System.out.println(stringList);
     //收集到Set集合
     Set<String> stringSet = list.stream().filter(element -> "男".equals(element.split("-")[1]))
             .collect(Collectors.toSet());
     System.out.println(stringSet);
     //收集到Map集合，以名字为键，年龄为值
     Map<String, String> stringMap = list.stream().filter(element -> "男".equals(element.split("-")[1])).
             collect(Collectors.toMap(
                     //参数1：键的生成规则
                     //Function泛型一:表示流中每一个数据的类型
                     //泛型二：表示Map中键的数据类型
                     //方法apply的形参：流中的每个数据
                     //方法体:生成键的代码
                     //返回值：生成的键
                     new Function<String, String>() {
                         @Override
                         public String apply(String s) {
                             String name = s.split("-")[0];
                             return name;
                         }
                     },
                     //参数2：值的生成规则
                     //Function泛型一:表示流中每一个数据的类型
                     //泛型二：表示Map中键的数据类型
                     new Function<String, String>() {
                         @Override
                         public String apply(String s) {
                             String age = s.split("-")[2];
                             return age;
                         }
                     }));
     System.out.println(stringMap);
